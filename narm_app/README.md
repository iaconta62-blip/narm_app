1. Instale o Flutter SDK: https://flutter.dev/docs/get-started/install
2. No terminal, posicione-se na pasta do projeto:
   cd narm_app
3. Rode:
   flutter pub get

4. O APK gerado estará em: build/app/outputs/flutter-apk/app-release.apk
